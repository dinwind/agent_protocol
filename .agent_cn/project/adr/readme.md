# 业务架构决策记录 (ADR)

> 本目录记录项目的业务架构决策，与协议层 ADR (`meta/protocol-adr.md`) 分离。

---

## ADR 模板

创建新 ADR 时使用以下模板：

```markdown
# ADR-XXX: 决策标题

## 状态

草案 | 已批准 | 已废弃 | 已取代

## 日期

YYYY-MM-DD

## 背景

描述导致此决策的背景和问题。

## 决策

描述做出的决策和原因。

## 后果

描述此决策带来的影响，包括正面和负面的。

## 备选方案

列出考虑过但未采用的备选方案。
```

## 使用说明

1. **何时创建 ADR**：
   - 重大技术选型
   - 架构模式变更
   - 重要的设计决策
   - 有争议的决策

2. **命名规则**：
   - 文件名：`adr-xxx-kebab-case-title.md`
   - 序号：三位数字，从 001 开始

3. **状态流转**：
   - 草案 → 已批准 → (已废弃 | 已取代)
