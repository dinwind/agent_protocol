# 任务预检清单

> 开始任何开发任务前必须完成的检查项。

---

## 快速检查（每次任务前）

### 1. 上下文确认

- [ ] 已阅读相关需求/Issue
- [ ] 理解任务的验收标准
- [ ] 明确任务的边界和范围

### 2. 知识库检查

- [ ] 查阅 `bug-prevention.md` 相关章节
- [ ] 检查 `known-issues.md` 是否有相关问题
- [ ] 确认是否有相关的 ADR

### 3. 依赖确认

- [ ] 确认上游依赖是否就绪
- [ ] 确认是否会影响下游功能
- [ ] 确认测试环境是否可用

### 4. 风险评估

- [ ] 识别可能的风险点
- [ ] 准备回滚方案（如适用）
- [ ] 确认是否需要额外审批
